## Build Setup

``` bash
# install dependencies
$ npm run install

# serve with hot reload at localhost:3000
$ npm run dev

# serve API at localhost:3001
# cd ../back
$ node index.js

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
